🍫 Chocolate Sales Performance Dashboard

📌 Project Overview

This project focuses on analyzing chocolate sales data using Power BI to gain insights into sales performance, profitability, and product-level trends.
It transforms raw data into an interactive dashboard that supports business decision-making.

---

🛠️ Tools & Technologies Used

- Power BI
- Excel / CSV Dataset

---

📂 Project Files

- "sample-chocolate-shipments-data-all-Apr-2025.xlsx" → Dataset
- "chocolate_sales_dashboard.pbix" → Power BI Dashboard
- "dashboard.png" → Dashboard Screenshot

---

📊 Dashboard Features

- KPI Cards: Total Sales, Total Profit, Profit %, Shipment Count
- Monthly Sales Trend Analysis
- Shipment Distribution Visualization
- Top Performing Products
- Top Salespersons Performance
- Team-wise Sales Contribution

🔍 Key Insights

- High-performing products contribute significantly to overall profit
- Sales show noticeable fluctuations across months
- Top salespersons generate a major share of total revenue
- Shipment distribution indicates varying demand patterns

🚀 Conclusion

This project demonstrates the use of Power BI to build interactive dashboards and extract meaningful insights from sales data.
It highlights skills in data visualization, business analysis, and dashboard design.
